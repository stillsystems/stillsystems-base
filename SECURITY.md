## Reporting a Vulnerability

Still Systems takes security seriously. If you find a security vulnerability in this project, please **do not open a public issue**. Instead, follow these steps:

1. **Email us**: Send a report to **security@stillsystems.org**.
2. **Details**: Include as much information as possible, including a proof of concept if available.
3. **Disclosure**: We will acknowledge your report within 48 hours and work with you on a disclosure timeline.

For more details, see our [Global Security Policy](https://github.com/stillsystems/.github/blob/main/SECURITY.md).
